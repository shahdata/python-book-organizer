# python-book-organizer
Python Project for organizing books, tracking reading lists and calculating book stats
Dataset:
Goodreads Listopia page (best Canadian literature):
https://www.goodreads.com/list/show/2702.Best_Canadian_Literature

Books Organizer and Rating Calculator
A Python-based interactive application that collects real book data from the web, allows users to build a custom reading list, and generates summary statistics such as average and median ratings.

This project was developed as part of a data-focused programming assignment to demonstrate skills in web scraping, data organization, menu-driven programming, and analytical reporting.

Project Overview:
A community library network is expanding a national Reading Challenge program.
Currently, staff manage book selections using spreadsheets, which leads to:

Duplicate book entries
Inconsistent ratings
Limited reporting and analysis

This project provides a Python prototype solution that:
Collects top books from a public website
Allows staff to build and manage a reading list through an interactive menu
Organizes book data reliably using aligned data structures
Generates summary reports by author
Calculates rating statistics (average and median)

The goal is to simplify reading list management and provide meaningful insights into book selections.

Features:
Web Data Collection
Retrieves book information from a public book list using web scraping
Parses HTML content using BeautifulSoup
Extracts:
Book titles
Authors
Ratings

Reading List Management

Interactive menu-driven interface
Allows users to:
View available books
Add books to their reading list
Display current selections

Rating Statistics:

Calculates
Average rating
Median rating

Provides quick insight into overall reading list quality

Author Summary
Groups books by author
Displays how many books from each author appear in the reading list

Robust Input Validation
Prevents crashes from invalid user input
Ensures reliable interaction

Technologies Used
Python
Requests – for web data collection
BeautifulSoup4 – for HTML parsing
Tabulate – for formatted table display
Termcolor – for colored terminal output
Regex (re) – for text extraction and cleaning

Installation:

Clone the repository

git clone https://github.com/shahdata/python-book-organizer

Navigate to the project folder

cd Python_BookPlanner

Install required dependencies

pip install requests beautifulsoup4 termcolor tabulate

How to Run the Program

Run the Python script:

python Python_BookPlanner.ipynb

You will see an interactive menu where you can:

1. Display available books
2. Add a book to the reading list
3. View reading list
4. Show rating statistics
5. Show summary by author
6. Exit
